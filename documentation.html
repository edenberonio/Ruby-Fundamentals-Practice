Ruby Documentation Review

STRINGS

  1. length: Returns the number of elements in self. May be zero.
  2. strip: Removes leading and trailing whitespace from str. Returns nil if str was not altered. EX: " hello ".strip #=> "hello"
  3. split: Divides str into substrings based on a delimiter, returning an array of these substrings. EX: " now's the time".split #=> ["now's", "the", "time"] or " now's the time".split(/ /) #=> ["", "now's", "", "the", "time"]
  4. start_with?: Returns true if str starts with one of the prefixes given. EX:"hello".start_with?("hell") #=> true

Array

  1. first: Returns the first element, or the first n elements, of the enumerable. If the enumerable is empty, the first form returns nil, and the second form returns an empty array.
  2. delete_at: Deletes the element at the specified index, returning that element, or nil if the index is out of range. EX:a = ["ant", "bat", "cat", "dog"] a.delete_at(2) #=> "cat"
  3. delete: Deletes all items from self that are equal to obj.
  4. pop: Removes the last element from self and returns it, or nil if the array is empty.

Time

  1. now: Alias for ::new. Returns a Time object initialized to the current system time.Creates a Time object based on given values, interpreted as UTC (GMT). The year must be specified. Other values default to the minimum value for that field (and may be nil or omitted). Months may be specified by numbers from 1 to 12, or by the three-letter English month names. Hours are specified on a 24-hour clock (0..23). Raises an ArgumentError if any values are out of range. Will also accept ten arguments in the order output by #to_a. EX: Time.utc(2000,"jan",1,20,15,1) #=> 2000-01-01 20:15:01 UTC Time.gm(2000,"jan",1,20,15,1) #=> 2000-01-01 20:15:01 UTC

File

  1. exist: Return true if the named file exists.
  2. extname: Returns the extension (the portion of file name in path after the period.) EX: File.extname("test.rb") #=> ".rb" File.extname("a/b/d/test.rb") #=> ".rb"
